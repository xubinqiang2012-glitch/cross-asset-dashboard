⚠ CORS 错误？使用本地服务器或 Python 脚本
浏览器从 file:// 直接拉取可能被 CORS 拦截。两种解决：
方案1 · 终端切到本目录后跑 python3 -m http.server 8000，浏览器访问 http://localhost:8000/gold-soros-dashboard.html
方案2 · 用同目录的 update_data.py 服务端拉取，写入 gold-data.json，仪表板自动读取（可加 cron 每周自动跑）
每周更新一次。数据保存在浏览器本地（localStorage）。建议固定每周一晚间 / 周二早间录入上一周收盘数据。
